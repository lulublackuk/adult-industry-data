- **Market Insights:** Booking patterns, rate calculations, demographics
- **Methodology:** Data collection and anonymization process

## Methodology

**Data Source:** Verified profiles on [Lulu Black Directory](https://lulublack.co.uk)

**Sample Size:** 21 verified providers

**Verification:** All providers manually verified by directory administrators
